# Diary Project
Es un proyecto de una aplicación que permite a los usuarios registrar medios (películas, series, videojuegos, libros, música), calificarlos y escribir reseñas personales sobre los mismos.

## Tecnologías utilizadas

### Frontend:

 - React
 - Vite
 - TailwindCSS
 - Chart.js
 - React Router

### Backend

 - Node.js
 - Express
 - SQLite
 - Typescript (tsx)
 - Nodemon
 - CORS
 - bcrypt
 - jsonwebtoken

## Requisitos previos
Antes de ejecutar el proyecto se necesita:
 - Node.js (versión 18 o superior)
 - npm
 - Git

## Instalación
Clonar el repositorio:

```bash
git clone <https://github.com/Nereag443/proyecto.git>
cd diary-project
```

## Cómo usar el proyecto
Una vez iniciado el proyecto:
 1. Abre el frontend en el navegador (por defecto http://localhost:5173)
 2. Regístrate o inicia sesión 
 3. Añade nuevos medios
 4. Asigna una puntuación y escribe una reseña
 5. Consulta tus estadíticas en la página de estadíticas 

## Ejemplos de uso

 1. El usuario registra varias películas y series durante el mes
 2. Asigna puntuaciones y escribe reseñas personales
 3. El sistema analiza los datos registrados
 4. Se generan gráficos que muestran la cantidad de medios consumidos por tipo.

## Funcionalidades principales

### Gestión de medios 
 - Creac, editar y eliminar registros de medios
 - Clasificación por tipo (película, serie, libro, etc.)

### Sistema de valoración
 - Puntuación personalizada por medio
 - Registro de reseñas detalladas
