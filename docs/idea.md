## Diario de consumo cultural
Diario personal en el que el usuario registra su actividad cultural (libros leídos, películas vistas, videojuegos jugados, música escuchada, etc).
Por cada consumo se podrá añadir una reseña y calificación.
Habrá diagramas de estadísticas y un resumen de media consumida (por semana, mes, año).

----

### Problema que intenta resolver

Con esta app se tendría un diario de consumo unificado, no simplemente como los clásicos que normalmente tienen solo un público en específico.

----

### Usuario objetivo

El usuario al que va dirigido este proyecto es un usuario que consume medios de diferentes tipo a diario y que disfruta de tenerlos registrados, además de reseñarlos.

----

### Funcionalidades principales

 - El usuario podrá añadir el medio que prefiera (libros, películas, videojuegos, música de forma predeterminada).

 - A partir de los medios introducidos, el usuario tendrá acceso a un sistema de estadísticas en el que aparecen los medios más consumidos y la cantidad

 - El usuario también podrá calificarlo mediante un sistema de calificación de estrellas

 - La aplicación también contará con un registro de usuario

----

### Funcionalidades opcionales

 - Sistema de rachas según la actividad del usuario
 - Sistema de niveles
 - Sistema reseñas públicas para que el usuario pueda compartir su opinión con el resto de usuarios


----

### Posibles mejoras futuras
